# Rusty Wiki

Markdown based wiki written in Rust.

Check out [Demo](https://sak96.github.io/rusty_wiki/) Editor.

  - [x] markdown live preview

- [ ] File system manager
  - [ ] store files
  - [ ] handle images
  - [ ] read files/directory (default index.md)
  - [ ] navigation view

## How to run web Editor ?

```bash
cargo install trunk
trunk serve
```
